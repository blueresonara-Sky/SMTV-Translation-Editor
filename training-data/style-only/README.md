This folder is a clean style-only dataset built from `D:\Downloads\Translation`.

Included files:
- Plain final Persian subtitle `.docx` files whose names end in `PER` or `PERSIAN`.
- Only one canonical copy was kept when both a plain file and a `(1)` duplicate existed.

Excluded files:
- `original`
- `fix` / `fixed`
- `rearranged`
- `Arranged by me`
- `gpt`
- `ChatGPT_API`
- `perplex`
- other experiment or generated variants

Structure:
- `item-###/final.docx`
- `index.json`

`index.json` maps each numbered item back to the original filename and source path.
